SmartHome Builder is a Java application designed to demonstrate core object‑oriented programming concepts through a simple, realistic home‑building model. The project fulfills all Week 5 Final Practical Exam requirements by implementing:

Inheritance (House extends Building)

Abstract Classes (Building)

Polymorphism (overridden getInfo() method)

Composition (House contains Door and Kitchen)

Constructors (all classes use parameterized constructors)

Access Specifiers (private, protected, public)

Text File Storage & Retrieval (log file creation, writing, and reading)

Iterative & Recursive Processing (printing building list both ways)
